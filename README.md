# Empire World Cleaning Dashboard

Live cleaning department dashboard for managing daily cleaning reports across 5 projects.

## Features

- 📊 **5 Projects**: Empire Complex, Empire Square, West Diamond, West Wing, Royal Apartment
- 🏢 **Building Management**: Track cleaning for 60+ buildings
- 📸 **Photo Documentation**: Paste images directly into reports
- 📅 **Date Filtering**: View reports by specific dates
- 📈 **Analytics**: Complete project statistics and building information
- 💾 **Data Persistence**: All data saved locally in browser
- 📥 **CSV Export**: Download analytics reports

## How to Use

1. Click on any project tab to add a daily cleaning report
2. Select building, enter employee count, pick floors to clean
3. Paste a photo (Ctrl+V) and choose cleaning level
4. Click "Save Report" to store
5. Use Analytics tab to view all project statistics
6. Share the link with your team!

## Setup (One-time)

1. Go to https://github.com and sign up (free)
2. Create a new repository named `empire-dashboard`
3. Upload the `index.html` file to the repository
4. Go to Settings → Pages
5. Select "Deploy from a branch" → "main" → "/(root)"
6. Save - your dashboard will be live at:
   **https://yourusername.github.io/empire-dashboard**

## Share the Link

Send this link to supervisors, team members, or anyone who needs access:
```
https://yourusername.github.io/empire-dashboard
```

They can open it in any browser on any device. All data is stored locally on their device.

## Data Management

- **View Analytics**: Click "📈 Analytics" tab
- **Filter by Date**: Use "📅 Dashboard" tab
- **Download Data**: Click "📥 Download Report" in Analytics
- **Reset Data**: Click "🔄 Reset Data" (deletes all local data)

## Support

All data is stored in your browser's local storage. Clear browser data/cache will delete saved reports. Backup your data regularly by downloading CSV reports.
